About

Send2Figma
for the handoff that never quite worked

We build tools for people who live in the browser, design in Figma, and ship in Cursor— without flattening the web into a screenshot.

GitHub See how it works

Why we exist

Capturing a live page usually means a flat image or a messy DOM dump. Designers lose editability. Engineers lose structure. Send2Figma sits in Chrome, captures what you see, and pastes editable layers into Figma—optionally with tokens and MCP for Cursor.

The goal is simple: keep fidelity high, keep the workflow short, and stay open enough that you can inspect and extend it.

What we care about

  1. 01

    Editable over pretty-but-dead

    Layers, text, and structure beat a single PNG every time.

  2. 02

    Browser-native capture

    Work from the live page—presets, select, design system, screenshot—without leaving Chrome.

  3. 03

    Figma and Cursor, not either/or

    Paste into Figma Desktop, or clone via MCP when you want code next to design.

  4. 04

    Open by default

    The extension and MCP live on GitHub so you can install, fork, and contribute.

Open source on GitHub
