How it works

From live Chrome to editable Figma in three steps.

Capture the viewport or a single element, pick Editable or Exact fidelity, then paste into Figma Desktop as layers you can restyle — or export tokens and clone via MCP.

  • Open any http(s) page and launch the Send2Figma dock
  • Capture with presets, custom width, or element select
  • Paste in Figma Desktop — or export tokens / clone via MCP
